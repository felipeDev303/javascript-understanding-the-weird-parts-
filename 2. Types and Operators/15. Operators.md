# Operators

## Definición

Símbolos especiales que realizan operaciones sobre uno o más valores (operandos) y devuelven un nuevo valor.

**Ejemplo:**
```js
3 + 2;  // el operador "+" conecta 3 y 2 → devuelve 5
```

---

## Tipos principales de operadores

1. **Aritméticos**
2. **Asignación**
3. **Comparación**
4. **Lógicos**
5. **String / Concatenación**
6. **Ternario**
7. **Type operators** (`typeof`, `instanceof`)

---

## 1. Operadores Aritméticos

| Operador | Ejemplo | Función |
|----------|---------|---------|
| `+` | 3 + 2 | Suma |
| `-` | 3 - 2 | Resta |
| `*` | 3 * 2 | Multiplicación |
| `/` | 6 / 2 | División |
| `%` | 7 % 3 | Resto |
| `**` | 2 ** 3 | Potencia |

**Ejemplo:**
```js
10 % 4;  // 2
```

---

## 2. Operadores de Asignación

| Operador | Significado |
|----------|-------------|
| `=` | asignación |
| `+=` | suma y asigna |
| `-=` | resta y asigna |
| `*=` | multiplica y asigna |

**Ejemplo:**
```js
let x = 10;
x += 5;  // x = x + 5
```

---

## 3. Operadores de Comparación

### Comparación "floja"
```js
==   // compara valor
!=   // distinto por valor
```

**Ejemplo:**
```js
"5" == 5;  // true (coerción)
```

### Comparación estricta (recomendada)
```js
===  // compara valor y tipo
!==  // distinto en valor o tipo
```

**Ejemplo:**
```js
"5" === 5;  // false
```

---

## 4. Operadores Lógicos

| Operador | Significado |
|----------|-------------|
| `&&` | AND lógico |
| `\|\|` | OR lógico |
| `!` | NOT lógico |

**Ejemplos:**
```js
true && false;  // false
true || false;  // true
!true;          // false
```

**Importante en JS:**  
`&&` y `||` no devuelven true/false, sino uno de los valores evaluados.

```js
console.log("JS" && 42);     // 42
console.log("" || "fallback");  // "fallback"
```

---

## 5. Operadores de String

Si uno de los operandos es string, `+` se convierte en concatenador:

```js
"Hola" + " " + "Mundo";  // "Hola Mundo"
5 + "5";                 // "55"
```

---

## 6. Operador Ternario

```js
condición ? valorSiTrue : valorSiFalse
```

**Ejemplo:**
```js
let edad = 18;
let mensaje = edad >= 18 ? "Mayor de edad" : "Menor de edad";
```

---

## 7. Type Operators

**`typeof`:**
```js
typeof 42;        // "number"
typeof "hola";    // "string"
typeof true;      // "boolean"
```

**`instanceof`:**
```js
[] instanceof Array;  // true
{} instanceof Object; // true
```
