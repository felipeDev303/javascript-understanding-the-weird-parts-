# Dynamic Typing

## Definición

JavaScript es un lenguaje de tipado dinámico: las variables NO tienen tipo, los valores sí.

**Diferencia con lenguajes tipados:**

**Java/C#:**

```java
int x = 10;  // x DEBE ser siempre un int
```

**JavaScript:**

```js
let x = 10; // number
x = "hola"; // ahora es string
x = true; // ahora es boolean
```

La misma variable puede contener valores de cualquier tipo.

---

## Dónde ocurre el tipado dinámico

En el momento de asignar el valor.

```js
let x = 3; // JS crea la variable, el objeto number 3 tiene tipo
x = "3"; // ahora contiene un string
```

No hay error porque la variable no tiene tipo fijo.

---

## Consecuencia: Type Coercion

Dynamic typing permite que JS intente convertir valores automáticamente:

```js
"5" * 2; // 10
"5" + 2; // "52"
true + 1; // 2
```

JavaScript interpreta los tipos según el contexto.

---

## Dynamic Typing vs Weak Typing

**Diferencia:**

- **Dynamic Typing:** tipo decidido en tiempo de ejecución
- **Weak Typing:** permite convertir tipos de forma implícita

JavaScript tiene ambas características.

**Ejemplos:**

```js
"10" - 1; // 9
"10" + 1; // "101"
```

En `+` da prioridad a strings, en `-` da prioridad a números.

---

## Ventajas

- Rápido de escribir
- Flexible
- Ideal para prototipado
- Perfecto para JSON y datos cambiantes
- Permite funciones genéricas

**Ejemplo:**

```js
function logger(x) {
  console.log(x); // funciona con cualquier tipo
}
```

---

## Desventajas

- Introduce errores difíciles de detectar
- Las funciones pueden recibir tipos inesperados
- Bugs silenciosos por coerción
- Menor predictibilidad
- Requiere más disciplina

**Ejemplo:**

```js
function sum(a, b) {
  return a + b;
}

sum("1", 2); // "12" (bug oculto)
```

---

## Sistema de tipos de JavaScript

**Tipos primitivos:**

- `number`
- `string`
- `boolean`
- `undefined`
- `null`
- `bigint`
- `symbol`

**Todo lo demás es object:**

```js
typeof []; // "object"
typeof {}; // "object"
typeof null; // "object" (quirk histórico)
typeof (() => {}); // "function"
```
