# Object Literals

## Definición

Forma más simple y directa de crear objetos en JavaScript usando la notación de llaves `{}`.

---

## Sintaxis básica

```js
let persona = {
  nombre: "Felipe",
  edad: 28,
  ciudad: "Madrid",
};
```

**Estructura:**

```js
{
  clave: valor,
  clave2: valor2
}
```

---

## Propiedades

### Tipos de valores

Las propiedades pueden ser cualquier tipo:

```js
let objeto = {
  string: "texto",
  numero: 42,
  booleano: true,
  array: [1, 2, 3],
  objeto: { interno: "valor" },
  funcion: function () {
    console.log("Hola");
  },
};
```

### Nombres de propiedades

**Válidos:**

```js
let obj = {
  nombre: "valor",
  _privado: "valor",
  $especial: "valor",
  123: "valor", // se convierte a string
  "con espacios": "valor",
};
```

**Acceso:**

```js
obj.nombre; // "valor"
obj["con espacios"]; // "valor"
obj[123]; // "valor"
```

---

## Métodos

Funciones dentro de objetos:

### Sintaxis tradicional

```js
let persona = {
  nombre: "Felipe",
  saludar: function () {
    console.log("Hola, soy " + this.nombre);
  },
};

persona.saludar(); // "Hola, soy Felipe"
```

### Sintaxis ES6 (method shorthand)

```js
let persona = {
  nombre: "Felipe",
  saludar() {
    console.log("Hola, soy " + this.nombre);
  },
};
```

---

## Características de ES6+

### 1. Property Shorthand

Si la variable y la propiedad tienen el mismo nombre:

```js
let nombre = "Felipe";
let edad = 28;

// Antes
let persona = {
  nombre: nombre,
  edad: edad,
};

// ES6
let persona = {
  nombre,
  edad,
};
```

### 2. Computed Property Names

Usar expresiones como nombres de propiedades:

```js
let prop = "edad";

let persona = {
  nombre: "Felipe",
  [prop]: 28,
  ["ciudad" + "Actual"]: "Madrid",
};

console.log(persona.edad); // 28
console.log(persona.ciudadActual); // "Madrid"
```

### 3. Spread Operator

Copiar propiedades de un objeto a otro:

```js
let persona = {
  nombre: "Felipe",
  edad: 28,
};

let empleado = {
  ...persona,
  cargo: "Developer",
};

console.log(empleado);
// {nombre: "Felipe", edad: 28, cargo: "Developer"}
```

**Sobrescribir propiedades:**

```js
let obj1 = { a: 1, b: 2 };
let obj2 = { ...obj1, b: 3 }; // {a: 1, b: 3}
```

## Objetos anidados

```js
let empresa = {
  nombre: "TechCorp",
  direccion: {
    calle: "Gran Vía",
    numero: 123,
    ciudad: "Madrid",
  },
  empleados: [
    { nombre: "Ana", cargo: "Developer" },
    { nombre: "Luis", cargo: "Designer" },
  ],
};

console.log(empresa.direccion.ciudad); // "Madrid"
console.log(empresa.empleados[0].nombre); // "Ana"
```

## Desestructuración de objetos

Extraer propiedades en variables:

```js
let persona = {
  nombre: "Felipe",
  edad: 28,
  ciudad: "Madrid",
};

// Desestructuración
let { nombre, edad } = persona;
console.log(nombre); // "Felipe"
console.log(edad); // 28

// Con alias
let { nombre: n, edad: e } = persona;
console.log(n); // "Felipe"

// Con defaults
let { nombre, pais = "España" } = persona;
console.log(pais); // "España"
```

## Object vs Object Literal

| Concepto               | Descripción                      |
| ---------------------- | -------------------------------- |
| **Object Literal**     | Sintaxis `{}` para crear objetos |
| **Object Constructor** | `new Object()` (menos usado)     |

**Preferir Object Literals:**

```js
// Bueno
let obj = {};

// Evitar
let obj = new Object();
```

---

## Modificar objetos

### Agregar propiedades

```js
let persona = { nombre: "Felipe" };
persona.edad = 28;
persona["ciudad"] = "Madrid";
```

### Eliminar propiedades

```js
let persona = { nombre: "Felipe", edad: 28 };
delete persona.edad;
console.log(persona); // {nombre: "Felipe"}
```

### Verificar existencia

```js
let persona = { nombre: "Felipe" };

"nombre" in persona; // true
"edad" in persona; // false

persona.hasOwnProperty("nombre"); // true
```

## Object.freeze() y Object.seal()

### Object.freeze()

Hace el objeto inmutable:

```js
let obj = { a: 1 };
Object.freeze(obj);
obj.a = 2; // No cambia
obj.b = 3; // No se agrega
delete obj.a; // No se elimina
console.log(obj); // {a: 1}
```

### Object.seal()

Permite modificar propiedades existentes, pero no agregar/eliminar:

```js
let obj = { a: 1 };
Object.seal(obj);
obj.a = 2; // Funciona
obj.b = 3; // No se agrega
delete obj.a; // No se elimina
console.log(obj); // {a: 2}
```

---

## Resumen

**Object Literals:**

- Sintaxis: `{clave: valor}`
- Forma más común de crear objetos
- Soportan propiedades, métodos, anidamiento
- ES6 añade: shorthand, computed properties, spread operator
- Preferir sobre `new Object()`
